中長距離アスリートのための練習日誌＆意見交換サービス

  • 新規登録
  • ログイン

About

記録を狙うランナーにとって、試合や練習の結果を記録し、分析することは非常に重要です。練習日誌(仮)はそんなランナー達をサポートする練習日誌&意見交換サービスです。

  • 今すぐ練習日誌を書く

機能

見やすいマイページ機能

自分の練習日誌をカレンダー形式で表示。次のレースまであと何日か、自分が過去にどんな練習をしてきたのかが見やすく表示されます。。

  • マイページへ

豊富な記録項目

天気や練習タイム、その日の反省等の基本的な項目はもちろん、食事内容、睡眠時間、満足度等様々な項目を記録可能。自分で追加もできます。

  • 練習日誌を書く

データをグラフで分析

これまでに記録した練習日誌のデータを見やすく可視化。日ごとの体重変化から月間走行距離の推移まで、可視化することで自分の成長が一目でわかります。
